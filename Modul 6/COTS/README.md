# Tugas COTS - Web CRUD Produk

## Deskripsi
Pada tugas ini dibuat sebuah halaman web sederhana untuk menampilkan dan mengelola data produk. Halaman web memiliki form input produk dan tabel data produk yang menggunakan JQuery DataTables.

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- Bootstrap
- JQuery
- DataTables

## Fitur Sistem
1. Form input data produk
2. Menampilkan data produk dalam tabel
3. Fitur pencarian (search)
4. Pagination pada tabel
5. Tombol hapus data
6. CRUD sederhana menggunakan object mapping
7. Penyimpanan data menggunakan LocalStorage

## Struktur Folder
COTS
- index.html
- css/style.css
- js/script.js

## Cara Menjalankan Program
1. Buka file `index.html` di browser
2. Masukkan data produk pada form
3. Klik tombol tambah produk
4. Data akan tampil di tabel
5. Data dapat dihapus menggunakan tombol hapus

## Kesimpulan
Program berhasil membuat sistem sederhana untuk mengelola data produk menggunakan teknologi web dasar dengan tampilan Bootstrap dan fitur tabel dari JQuery DataTables.
